# Claude Guides

A collection of user guides for Anthropic's Claude products.

## Available Guides

### [Cowork](./cowork/Claude_Desktop_Cowork_User_Guide.md)

A comprehensive guide to Claude Desktop's Cowork mode, covering everything from first launch to advanced automation. Topics include folder setup, task delegation, file creation (Word, Excel, PowerPoint, PDF), plugins, shortcuts, scheduling, and troubleshooting.

### [Claude Agent in Xcode 26.3](./claude-in-xcode/Claude_Agent_Xcode_26.3_Advanced_Developer_Guide.md)

An advanced developer guide to agentic coding with Claude Agent in Xcode 26.3. Covers the Claude Agent SDK integration, Model Context Protocol, autonomous AI workflows, and best practices for building, testing, and shipping Apple platform apps faster.

### [AI-Assisted iOS Development](./ai-ios/AI-iOS-Development-Guide.md)

A step-by-step guide to building a complete iOS app using an AI-assisted workflow with Cowork, Claude Agent in Xcode, and GitHub. Follows a concrete example project (StreakPal, a habit tracker) from idea through App Store submission using SwiftUI, SwiftData, and MVVM architecture.
